# Bimal Fortnite Site

Welcome to Bimal's Fortnite Site! This website showcases various aspects of Fortnite, including Points of Interest (POIs), featured videos, and information about Bimal Adhikari and his friends.

## Project Structure

- `/index.html`: The main landing page of the website.
- `/css/`: Directory containing all CSS files for styling the website.
- `/css/styles.css`: Main stylesheet for the website.
- `/js/`: Directory containing all JavaScript files for the website.
- `/images/`: Directory containing all images used on the website.

## Features

### Home Page (index.html)
- Displays the latest news and featured Fortnite gameplay videos.
- Videos are embedded using iframes.

### About Page (HTML/about.html)
- Provides information about Bimal Adhikari.
- Displays a list of Bimal's friends with their images and descriptions.

### Contact Page (HTML/contact.html)
- Contains a contact form for users to get in touch.
- Form validation is implemented using JavaScript.

### POI Page (HTML/POI.html)
- Lists various Points of Interest (POIs) in Fortnite.
- Each POI includes an image, name, and description.

## Styles
- The website uses a single CSS file located at `CSS/styling.css` for styling.

## JavaScript
- `JavaScript/about.js`: Handles the dynamic creation of friend cards on the About page.
- `JavaScript/contact.js`: Implements form validation for the Contact page.
- `JavaScript/poi.js`: Handles the dynamic creation of POI cards on the POI page.

## How to Run
1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the home page.
3. Navigate through the website using the navigation links to explore different sections.
